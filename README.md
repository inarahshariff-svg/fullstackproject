# fullstackproject
SERS (System for Emotional Response and Analysis) is a classroom emotion monitoring platform that helps teachers track students’ emotional states in real time. It uses webcam-based emotion detection to identify moods like happy, sad, angry, fearful, or neutral, and classifies students into alert levels such as green, yellow, and red. The system also provides teacher coaching suggestions, student emotion trends, session history, and parent notifications when prolonged distress is detected. Its main goal is to support student well-being by helping teachers respond early and appropriately.
